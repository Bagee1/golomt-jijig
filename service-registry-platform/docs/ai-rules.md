# AI Usage Rules

- Keep AI tasks small and scoped.
- Read existing files before editing.
- Do not remove PDF-required fields.
- Do not store passwords as plain text.
- Do not commit `.env` or secrets.
- Do not add enterprise features before MVP works.
- Run tests/build after code changes.

Task order:

1. Database and backend skeleton.
2. Auth.
3. System CRUD.
4. List/search.
5. Security checklist.
6. Audit log.
7. Frontend.
8. Banking transfer demo.
